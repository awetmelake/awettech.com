Movie DB

React / Redux / Sass / Firebase / OpenMovieDatabaseAPI

remove_red_eye demo code GitHub
React application that uses the Open Movie Database Api to let users search from 1000's of media titles. Search movies, tv-shows, and games. Click on posters to be taken to details page where the title's information such as plot and imbd rating is displayed. Users can add and remove titles from their watchlist when signed in. Changes are updated live with Cloud Firestore. Uses Firebase for authentication and Firestore for storing users movie collections. React router handles routing and 404 pages. Redux used for state management.

Taskflow

React / Redux / Firebase / React-redux-firebase / Redux-firestore / Materialize UI

remove_red_eye demo code GitHub
React task management app similar to Kanban / Jira. Users can sign in and create 'task boards' that contain panels that contain tasks. Customize boards, panels, and tasks. Create tasks with custom descriptions and colors signifying whatever you want then to, ability to filter tasks by color. Also includes the ability to select a task and set a timer - pomodoro/stopwatch. When the timer is stopped the elapsed time is added to the tasks "time spent" property, which you can see when you click on the task. Uses Google Firebase for authentication and Cloud Firestore for storing user info with realtime update functionality.

Tetris.js

JS / HTML / CSS / Bootstrap / Sass / Webpack

remove_red_eye demo code GitHub
The clasic title: Tetris, made with JS using MVC and OOP design patterns. The project's code is organized according to MVC. There is a Controller class that sets event listeners and processes user actions to actions with in the Game class. A Game class that defines the logic of the game. Consists of 2d arrays that represent the cells of the board matrix. A display file that contains the HTML canvas display that draws the visual representaion of the Game matrix as its changing. Weather App

React / Redux/ Open Weather Map API / Bootstrap

remove_red_eye demo code GitHub
React application that uses the Open Weather Map API to let users get the forecast for their area based on browser navigation object, or user inputted zip code. Display's current forecast, wind speed, and various other info as well as icons signifying the conditions. Gives option to toggle metric/imperial. Hourly forecast display for future days are shown on a carousel, with the ability to click on individual times and get a detailed forecast report of it.

Postr

PHP / MySQL / JS / Materialize.css

remove_red_eye demo code GitHub
PHP/MySQL job posting board. Browse from job postings created by users. Sign in to create job posts, or save them from your favorites. Users can edit or delete postings that that have created, and save posts that other users have created. User auth handled through session super globals.
