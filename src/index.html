Taskflow

React / Redux / Firebase / React-redux-firebase / Redux-firestore / Materialize UI

remove_red_eye demo code GitHub
React task management app similar Kanban / Jira. Users can sign in and create 'task boards' that contain panels that contain tasks. Customize boards, panels, and tasks. Create tasks with custom descriptions and colors signifying whatever you want then to, ability to filter tasks by color. Also includes the ability to select a task and set a timer - pomodoro/stopwatch. When the timer is stopped the elapsed time is added to the tasks "time spent" property, which you can see when you click on the task. Uses Google Firebase for authentication and Cloud Firestore for storing user info with realtime update functionality.

Weather App

React / Redux/ Open Weather Map API / Bootstrap

remove_red_eye demo code GitHub
React application that uses the Open Weather Map API to let users get the forecast for their area based on browser navigation object, or user inputted zip code. Display's current forecast, wind speed, and various other info as well as icons signifying the conditions. Gives option to toggle metric/imperial. Hourly forecast display for future days are shown on a carousel, with the ability to click on individual times and get a detailed forecast report of it.

Postr

PHP / MySQL / JS / Materialize.css

remove_red_eye demo code GitHub
PHP/MySQL job posting board. Browse from job postings created by users. Sign in to create job posts, or save them from your favorites. Users can edit or delete postings that that have created, and save posts that other users have created. User auth handled through session super globals. Data stored in a MariaDB server through my hosting provider. This project was made for the purpose of showcasing my ability to learn new technology and apply it quickly. I created this project in under 2 weeks with no prior PHP or MYSQL knowledge.
