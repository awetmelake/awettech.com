Movie DB

See live GitHub
React application that uses the Open Movie Database Api to let users search from 1000's of media titles. Search movies, tv-shows, and games. Click on posters to be taken to details page where the title's information such as plot and imbd rating is displayed. Users can add and remove titles from their watchlist when signed in. Changes are updated live with Cloud Firestore. Uses Firebase for authentication and Firestore for storing users movie collections. React router handles routing and 404 pages. Redux used for state management.

Taskflow

See live GitHub
React app that lets users organize their tasks. Users can sign in and create task boards that contain sections/panels that contain tasks. Customize board, panel, and task names and their order. Create tasks with custom descriptions and colors signifying whatever you want then to, ability to filter tasks by color. Also includes the ability to select a task and set a timer - pomodoro/stopwatch. When the timer is stopped the elapsed time is added to the tasks "time spent" property, which you can see when you click on the task, for time tracking. Uses Firebase for authentication and Firestore for storing user info with realtime update functionality. React router handles routing and 404 pages.

Tetris.js

See live GitHub
My first ever web project - Tetris. Made with vanilla JS using the MVC design. The project's code is seperated into 3 files: game logic, display and controller logic, corresponding to the seperate elements of the MVC framework. Order of operations: the controller class invokes methods within the game class that change data such as the values in the 2d array of the board matrix and the orientation of the current peice being controlled by the player, the display file contains the HTML canvas display that draws the visual representaion of this data as its changing.

Weather App


See live GitHub
React application that uses the Open Weather Map API to let users get the forecast for their area based on browser navigation object, or user inputted zip code. Display's current forecast, wind speed, and various other info as well as icons signifying the conditions. Gives option to toggle metric/imperial. Hourly forecast display for future days are shown on a carousel, with the ability to click on individual times and get a detailed forecast report of it.
